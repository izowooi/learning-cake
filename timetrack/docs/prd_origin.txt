팀원들의 일정관리를 편하게 하기 위해, 그리고 시각적으로 보기를 원합니다.
팀원마다 업무가 있으며 이 업무는 짧게는 하루, 길게는 수일 동안 지속됩니다.
특히 어떤 팀원의 업무가 어디까지 진행되어있고, 이 업무가 다른 팀원에게 영향이 가기 때문에 시각적으로 볼 필요가 있습니다.
그래서 팀원들의 간트 차트 웹앱을 만드려고 합니다.
모바일에서, 데스크탑에서 (맥, 윈도우) 잘 동작해야합니다.
팀원들의 숫자는 적게는 4명 많게는 20명까지 가능합니다.
첫 화면에서 팀의 이름을 입력하면 해당 팀의 간트 차트를 보여주세요.
이후에 재접속하면 마지막으로 확인한 팀의 간트 차트를 보여주세요.
팀원이 연차를 쓰거나, 휴가를 가면 이 내용을 반영할 수 있어야합니다. ( 연차를 사용하면 기존 업무가 딜레이 되거나 유지 됩니다. on/off )
팀원을 추가할 수 있어야하며, 팀원의 업무의 시작과 종료를 넣을 수 있어야합니다.
최소 업무 단위는 하루입니다.
firebase realtime db 를 통해 데이터를 저장합니다.
아래는 제가 미리 만들어놓은 frdb 입니다.
https://ganttly.asia-southeast1.firebasedatabase.app/
또한 더미로 미리 "ganttly" 키 값만 빈값으로 만들어뒀습니다.
이를 활용해서 데이터를 저장해주세요.
제 개인적은 생각에 ganttly 라는 키 값안에 각각의 팀 이름으로 딕셔너리를 만들어서 거기에 구성원을 넣으면 될것 같다고 생각합니다.
기술 스택 : React + Next.js, Tailwind css, Firebase Realtime DB, Cloudflare Pages 등
