
마니또 웹 앱을 만드려고 합니다.

먼저 이 서비스의 사용 방법을 설명합니다.
마니또 사이트에 접속합니다.
사이트에 접속하며 그룹명과, 리더의 이름을 입력합니다.
LeaderA, GroudA 를 넣었다고 가정합니다.
그룹생성을 누릅니다. 다음 페이지로 넘어갑니다. ( 일치하면 일치하는 리더, 그룹이 있다고 알려주고, 진행하지 마세요 )
다음페이지에 넘어가면 "어느 그룹"의 "어느 리더" 인지 UI 로 표시하고, 그룹의 비밀번호를 임의로 생성해주세요.
그룹의 비밀번호는 "형용사+명사"의 형식으로 한글로 미리 json 데이터를 만들어두고, 여기서 참고하세요.
다음 페이지는 이름을 입력하는 페이지입니다. 여기서는 이름을 입력하고 "이름추가"를 누르면 아래에 추가된 이름이 표기됩니다.
A, B, C 를 넣었다고 가정합니다. ( 이 때 이름은 수십개가 될 수 있으니, 스크롤 될 수 있어야합니다 )
또한 각각의 이름 우측에는 변경하거나 지울수 있는 버튼이 이쁘게 보여야합니다.( 단순히 직관적인 아이콘만 보여주세요.)
이름을 각각 추가한 후에는 추가된 이름이 표기된 맨 아래에 "매칭 시작"을 누릅니다.
매칭 시작을 누르면 바로 매칭 결과 페이지가 표시됩니다.
여기에는 각각의 누가 누구에게 매칭되었는 지 표시합니다.

B ➡️ C

C ➡️ A
그리고 위의 결과는 맨처음에 접속하는 페이지에서 관리자 페이지로 이동하는 기능을 제공하여 볼 수도 있어야 합니다.
위에 설명한 대로 페이지는 총 4개가 있습니다.
메인 페이지 : 시작점
이름 넣는 페이지 : 여기서 이름을 넣고, 그룹의 암호를 보여줍니다.
결과 페이지 : 여기서 매칭 결과를 확인합니다.
관리자 페이지 : 여기서 매칭 결과를 확인하고, 관리할 수 있습니다.
일단은 제가 혼자 사용하기 위해서 DB 는 연동하지 말고 로직과 페이지를 모두 구성해주세요.
