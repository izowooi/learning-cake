이제 매칭이 완료되면 아래와 같이 매칭 비밀번호가 나와야합니다.
A -> B ( 매칭 비밀번호 : 화사한강아지 )
B -> C ( 매칭 비밀번호 : 느긋한독수리 )
C -> A ( 매칭 비밀번호 : 정직한늑대 )
이 비밀번호는 마니또 구성원이 본인의 마니또 매칭을 확인하는데 사용됩니다.
또한 해당 페이지에서는 후술할 미션의 내용이 포함됩니다.

이제 미션이 존재해야합니다. ( 미션은 On/Off 가 가능하다, 기본값은 Off )
미션은 제목, 내용, 점수로 구성된다. 점수는 100이하로 한다.
미션은 관리자가 맨 처음 마니또 그룹을 만들 때에만 추가할 수 있다. ( 최대 5개 )

또한 앞서말햇듯이 마니또 구성원(팀원)은 누구에게 매칭 되었는 지 확인이 가능해야합니다.
기존에는 그룹 비밀번호를 관리자가 입력하여 모든 마니또 구성원의 매칭 결과를 알 수 있었습니다.
이제 마니또 구성원은 매칭이 된 비밀번호를 부여받아 ( 그룹 비밀번호와 다릅니다. 후술 하겠습니다. )내가 어느 그룹에 속해있고, 어떤 사람이 대상인지 확인할 수 있어야합니다.
여기에는 리더가 생성한 마니또 그룹의 미션의 내용이 있어야 합니다.
또한 구성원은 해당 페이지가 본인이 수행한 미션을 수행했다고 이 페이지에 기입해야합니다. ( 현재는 텍스트만, 추후에 이미지 넣어야함 )

이제 마니또 그룹 관리자가 결과 페이지를 확인했을 때 모든 구성원이 수행한 미션이 나와야합니다.
여기에는 각 구성원이 작성한 텍스트와 미션의 점수 등이 나와야합니다.( 추후에 이미지 추가할 예정 )
